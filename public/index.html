Hyperspace Deck Command: Wishgranter

Wishgranter is a modloader for Sleeper Games' Hyperspace Deck Command. It works by loading the base games files with a seperate electron application, adding a number of hooks and changing the way loading resources works to allow definition chaining, then reading some number of mod files to add extra content.
